Once the input data has been collected, the bot must determine how the true probability of the outcome has changed. This is where Bayesian updating comes in:
The logic is straightforward. The bot does not blindly trust the current market price. It takes the previous probability, applies new evidence to it, and gets a new fair probability.

def bayes_update(prior, like_true, like_false):
    num = like_true * prior
    den = num + like_false * (1 - prior)
    return num / den

prior = 0.41
posterior = bayes_update(prior, 0.78, 0.24)
print(f"Updated probability: {posterior:.2%}")

def fractional_kelly(p, b, lam=0.25):
    f_star = (b * p - (1 - p)) / b
    return max(lam * f_star, 0)
